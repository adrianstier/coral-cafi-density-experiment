# Data Directory

This directory contains all data files for the MRB coral-CAFI analysis.

## Directory Structure

```
data/
├── MRB Amount/           # Raw data from field experiment
│   ├── CAFI community data
│   ├── Coral physiology data
│   ├── 3D photogrammetry measurements
│   └── Treatment assignments
│
├── processed/            # Processed data generated by analysis scripts
│   └── coral_growth.csv  # Growth metrics (from Script 6)
│
└── README.md            # This file
```

## Data Dictionary

### Treatment Data (`coral_id_position_treatment.csv`)

| Variable | Type | Description | Values |
|----------|------|-------------|--------|
| `coral_id` | string | Unique coral identifier | e.g., "FE-1", "FE-2", ... |
| `position` | integer | Position within reef patch | 1-6 |
| `treatment` | integer | Number of coral colonies per patch | 1, 3, or 6 |

### CAFI Community Data (`1. mrb_fe_cafi_summer_2021_v4_AP_updated_2024.csv`)

| Variable | Type | Description | Example |
|----------|------|-------------|---------|
| `coral_id` | string | Coral hosting this observation | "FE-1" |
| `date_collected` | date | Observation date | "2021-05-15" |
| `code` | string | Species code | "DAFLA" |
| `count` | integer | Number of individuals | 1-50 |
| `cafi_size_mm` | numeric | Body size (mm) | 15.5 |
| `lowest_level` | string | Taxonomic identification | "Dascyllus flavicaudus" |
| `phylum` | string | Taxonomic phylum | "Chordata" |
| `class` | string | Taxonomic class | "Actinopterygii" |
| `order` | string | Taxonomic order | "Perciformes" |
| `family` | string | Taxonomic family | "Pomacentridae" |

### Coral Physiology Data (`1. amount_master_phys_data_v5.csv`)

| Variable | Type | Units | Description |
|----------|------|-------|-------------|
| `coral_id` | string | - | Coral identifier |
| `protein_mg_cm2` | numeric | mg/cm² | Protein content per surface area |
| `carb_mg_cm2` | numeric | mg/cm² | Carbohydrate content per surface area |
| `zoox_cells_cm2` | numeric | cells/cm² | Zooxanthellae density |
| `afdw_mg_cm2` | numeric | mg/cm² | Ash-free dry weight |
| `surface_area` | numeric | cm² | Coral surface area from wax dipping |

### 3D Photogrammetry Data (`MRB_*_mesh_measure.csv`)

| Variable | Type | Units | Description |
|----------|------|-------|-------------|
| `Chunk` | string | - | Coral identifier (matches coral_id) |
| `Surface Area (cm^2)` | numeric | cm² | 3D surface area from mesh |
| `Volume (cm^3)` | numeric | cm³ | Enclosed volume |
| `Max height (cm)` | numeric | cm | Maximum colony height |
| `Min height (cm)` | numeric | cm | Minimum height (above substrate) |

## Processed Data

### `processed/coral_growth.csv`

Generated by `scripts/MRB/6.coral-growth.R`. Contains growth metrics for 44 corals with ≥80% survival.

| Variable | Type | Units | Description |
|----------|------|-------|-------------|
| `coral_id` | string | - | Coral identifier |
| `treatment` | factor | - | Treatment group (1, 3, or 6) |
| `reef` | factor | - | Reef patch identifier |
| `percent_alive` | numeric | % | Proportion of live tissue |
| `volume_cm3_2019` | numeric | cm³ | Initial volume |
| `volume_cm3_2021` | numeric | cm³ | Final volume |
| `growth_vol_b` | numeric | cm³ | Size-corrected growth (unified allometric model) |

## Data Collection

### Study Site
- **Location:** Moorea, French Polynesia
- **Habitat:** Back reef lagoon
- **Depth:** 1-3 m

### Experimental Design
- **Species:** *Pocillopora* spp. (branching corals)
- **Treatments:** 1, 3, or 6 coral colonies per reef patch
- **Replicates:** 6 reef patches per treatment (18 total patches)
- **Duration:** May 2019 - May 2021 (2 years)

### Measurement Methods

| Data Type | Method | Reference |
|-----------|--------|-----------|
| 3D structure | Structure-from-motion photogrammetry | Agisoft Metashape |
| Physiology | Tissue slurry assays | Veal et al. 2010; Smith et al. 1985 |
| CAFI community | Visual census + collections | Standard protocols |

---

## Data Processing Notes

1. **CAFI filtering:** Species included if observed on ≥10 distinct colonies
2. **Survival threshold:** Corals with <80% live tissue excluded from most analyses
3. **Growth calculation:** Uses unified allometric model (pooled across treatments)
4. **Missing data:** Some physiology measurements missing due to sample loss
